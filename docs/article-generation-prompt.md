Use this prompt when generating new Chinese surname articles.

```text

Write an English SEO/GEO article using the site publishing standard.

- Search intent:
- Page type:

Writing rules:
- Open with a direct answer in the first paragraph.
- Separate character meaning from surname origin.
- Explain romanization carefully when spellings differ.

[Summarize and guide to next page/tool.]
```
