# Chinese Surname Guide 发文标准

> 适用范围：百家姓站所有 surname profiles、guides、reference lists、FAQ、后续新增文章页。

## 1. 发文前必须具备的信息

| 字段 | 标准 |
|---|---|
| Primary Keyword | 一个主关键词，不与已发布页面冲突 |
| Search Intent | general / reference-list / meaning-origin / pronunciation |
| URL Slug | 英文小写短横线，不堆词 |
| SEO Title | 35-78 字符，核心词靠前 |
| Meta Description | 90-165 字符，说明用户能得到什么 |
| Article Type | guide / surname-profile / list / meaning / origin / FAQ |
| Related Internal Links | 3-5 个候选内链 |
| FAQ Plan | 2-5 个当前页面专属 FAQ |

## 2. 写作原则

- 先回答，再解释。
- 区分 character meaning 和 surname origin。
- 解释 pinyin、Chinese character、romanized variants 三者关系。
- 不承诺验证个人家谱。
- 不把单个英文拼写直接等同唯一汉字。
- 对海外拼写要谨慎，例如 Lee、Wong、Ng 可能对应多个汉字或地区读音。

## 3. 正文结构

推荐结构：

```text
直接答案
-> 字符 / 拼音 / 常见拼写
-> 含义说明
-> 来源说明
-> 常见误区
-> Related Guides
-> FAQ
-> 结尾
```

## 4. SEO/GEO 标准

| 项目 | 标准 |
|---|---|
| H1 | 每页仅 1 个 |
| H2 | 至少 2 个 |
| FAQ | 详情页 2-5 个专属 FAQ |
| Canonical | 必须指向当前正式 URL |
| Sitemap | 发布后进入 sitemap |
| 内链 | 每篇 3-5 条 |
| 短答案 | 页面内至少 1 段可直接引用 |

## 5. 合规边界

- 不验证私人家谱。
- 不把姓氏含义写成个人命运、性格或身份判断。
- 不把一个英文拼写武断对应一个唯一汉字。
- 不写“guaranteed origin”“your family came from...”这类结论。
